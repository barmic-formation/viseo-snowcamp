API Stream

Un stream est un flot d'objects sur le quel on applique des traitements. API fluent inspirée des langages fonctionnels.

Programmation réactive

Paradigme consistant à modéliser un traitement comme un flot d'évènements modifiant l'état du système

  • Orienté message : chaque évènement est un nouveau message reçu
  • Asynchrone : les modifications peuvent produire de nouveaux évènements
